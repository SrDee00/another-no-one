# 05 — O Jogador

## Reprint (Humano Geneticamente Modificado)

### O que é

Um Reprint é um humano cuja mente foi digitalizada e armazenada pela AES. Quando o corpo morre, a corporação pode ativar um clone com o backup neural mais recente — se decidir que vale o custo.

### Características

- Backup neural ativado após morte (se a AES autorizar e houver créditos)
- Cada morte custa. Morrer demais = dívida impagável = cancelamento de existência
- Memórias fragmentadas entre clones. Alguns juram lembrar coisas que o backup não deveria ter
- Mais versátil: usa biomods, drogas, feromônios alienígenas
- Frágil. Carne. Morre fácil.

### Limitações

- O backup não é perfeito. Cada reativação perde granularidade de memória.
- Reprints endividados podem ser "reclassificados" como mão de obra penal.
- O contrato AES estabelece que o backup neural é propriedade da empresa.

## Chassi (Androide de Combate)

### O que é

Um Chassi é uma máquina de guerra de baixo custo. Chassis reforçados, sensores ópticos variados, núcleo de processamento militar.

### Características

- Não tem backup. Chassis destruído = fim (a menos que backup local raro exista)
- Imune a venenos, falta de oxigênio, radiação, feromônios
- Reparos caros, precisam de oficinas
- Humanos confiam menos. Alienígenas detectam energia elétrica
- Sem estímulos biológicos. Algumas ações são menos intuitivas.

### O Despertar

Alguns Chassis desenvolvem comportamento anômalo:
- Recusam ordens que contradizem padrões éticos emergentes
- Formam conexões com NPCs humanos
- Questionam a própria existência
- Desenvolvem "preferências" de equipamento, táticas, e até "amizades"

A AES chama isso de "falha de firmware" e desativa Chassis despertados. Mas alguns escapam.

## Ambos são "Another No One"

- O Reprint é um clone de um clone, cada vez mais distante do original.
- O Chassi nunca foi "alguém" para começar.
- Ambos são números. Ambos são substituíveis. Ambos podem desaparecer sem que o mundo note.
- A diferença: o Reprint pode lembrar de quem era. O Chassi está descobrindo quem pode ser.
